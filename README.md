hungry-irc
==========

Irc bot designed for working private networks

Requirements:
- Python 3.2

How to:

1. python create_db.py
2. python test_db.py
3. vi hungry.py and edit config
4. python hungry.py

Features:

- random phrases
- Coffee time!
- random places to eat

Future features:

- It will add and remove phrases
- It will add and remove places to eat
